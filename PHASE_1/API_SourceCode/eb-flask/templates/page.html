Disease Report API

Endpoints

GET

/getReport?id={id}&date={date}&country={country}&cases={cases}&disease={disease}

This endpoint retrieves disease reports from within the database that will correspond to the queries. Each query is optional and act as filters when searching for disease reports. Parsing no parameters will return all disease reports.

Example

Query: /getReport?country=Australia&disease=coronavirus

Return:

[{"id": 1, "date": "2021-2-1", "country": "Australia", "cases": 5, "disease": "coronavirus"},
{"id": 2, "date": "2021-2-24", "country": "Australia", "cases": 1, "disease": "coronavirus"}]

PUT

/addReport

This endpoint adds a disease report into the database. A corresponding JSON file must be passed in which it will be added to the database.

Example

Query: /addReport {"id": 1, "date": "2021-2-1", "country": "Australia", "cases": 5, "disease": "coronavirus"}

Return:

400 OK
